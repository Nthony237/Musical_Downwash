 Musical Downwash

## Dependencies
### Python Libraries
pip install -r requirements.txt

### External Repos (place alongside this folder)
- Crazyswarm2: https://github.com/IMRCLab/crazyswarm2
- Crazyflie Firmware: https://github.com/bitcraze/crazyflie-firmware

## Setup
1. Clone this repo
2. Clone the external repos above into the same parent directory
3. pip install -r requirements.txt
4. Run python3 analysis/beat_analysis.py to test audio analysis
